# Hope-you-never-need-it

## 关于

随着人类的发展，环境正在变得愈发糟糕。自然灾害频发的同时，各国之间也剑拔弩张。刚刚“过去”的“Covid-19”大流行，也许让不少人都意识到了应急求生知识的重要性。

本项目旨在创建一个免费、中立的应急小册。我们希望你永远用不到这本小册上的知识，如果遇到了我希望这本小册可以帮助你脱离困难，安全回家。
